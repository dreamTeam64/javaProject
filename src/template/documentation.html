Bonjour, merci d'utiliser notre application !

Voici un petit guide pour vous aider à utiliser au mieux notre application facilement.

Comment utiliser l'application:

  • Votre recherche

    • Naviguez sur le web et copiez l'url du flux rss, puis collez cette url dans le champs "votre lien rss:".

    Maintenant cliquez sur Pau'd'caster, si tout se passe bien vous devriez voir apparaitre les actualités sur la droite de la page !

  • Notre Selection !

    • Vous trouverez une selection de fluxs rss déjà présents dans notre application.

    Il vous suffit de cliquer sur une des images pour acceder à une page contenant toutes les emissions du flux choisi

  • Player, à droite de la page

    • C'est ici que vous retrouverez les actualités liées au flux que vous avez entré dans la barre de recherche !

  • Vos Favoris

    • Vous pouvez facilement garder vos émissions préférées à portée de main, pour cela rien de plus simple ! Vous entrez un lien et vous cliquez sur sauvegarder ! Une liste de vos favoris s'affichera, si vous cliquez sur un titre il se chargera dans le flux, si vous cliquez sur del il se retirera de vos favoris
